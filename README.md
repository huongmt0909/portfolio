## 🚀 Getting Started

```bash
# Clone repo
git clone https://github.com/huongmt0909/portfolio.git
cd portfolio

# Install dependencies
yarn install

# Run dev server
yarn dev
```

Open [http://localhost:3000](http://localhost:3000)

```bash
# Build production
yarn build
```
